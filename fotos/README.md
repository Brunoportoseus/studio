# Fotos — Denise de Paula

Pasta para upload de fotos do site.

## Como usar

1. Faça upload das fotos diretamente nesta pasta pelo GitHub
2. Após o upload, a URL pública de cada foto será:

```
https://brunoportoseus.github.io/studio/fotos/NOME-DA-FOTO.jpg
```

3. Cole essa URL no painel Admin → Imagens

## Dicas

- Formatos aceitos: `.jpg`, `.jpeg`, `.png`, `.webp`
- Para a Vitrine: fotos em formato retrato (vertical) ficam melhores
- Para o Hero: foto horizontal, preferencialmente com Denise à direita
- Renomeie os arquivos sem espaços ou acentos (ex: `sobrancelha-01.jpg`)
